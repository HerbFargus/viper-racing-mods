Short tutorial on how to make hittable walls for any Viper Racing track object like signs,buildings,guard rails ....etc....

Paste the wall.mod object you exported from Zmodeler or wherever here.
For example if you had a cube object like a house1.mod exported from zmodeler,you would first make a
copy of the house1.mod,name it wall-house1.mod,import it into zmodeler,delete the top and bottom
of the cube type house and export it into a mod2wall folder ( that has the mod2quadnoz.exe ) inside the
folder you are making the track in.

To make walls for any object they must be vertical only,must not be tilting and no horizontal parts will work.
To make walls you run the mod2quadnoz.exe program,
then set the top and bottom to the example numbers below:

+0 bottom and +1.0 top is best height for guard rails

+0 and +4.0 for big billboards

+0 and +2.0 for small billboards

+0 and +8 for start line banner posts

+0 and +10 for 1 story buildings

Next you click the process button and double click the wall mod file.
Copy the contents of the generated name-mod-noz.txt and paste it into your "foolandsurface.txt" file in the main track making folder.
Now when you put your track together and replace any track in Trackman,you will be able to hit that house,
not drive through it like before.
